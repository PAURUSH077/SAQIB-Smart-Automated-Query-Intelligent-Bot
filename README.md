# 🚀 SAQIB – Smart Automated Query & Intelligent Bot

> 🎤 Your own **AI-powered voice assistant** that understands natural language, automates tasks, and interacts like a human.

---

## 🧠 Overview

**SAQIB** is a voice-controlled AI assistant that combines:

* 🎤 Speech Recognition
* 🤖 AI Understanding (Gemini)
* ⚙️ System Automation
* 🌐 Browser Control (Selenium)

It can **listen, understand, decide, and act** — just like a mini Jarvis.

---

## ✨ Features

### 🎤 Voice Interaction

* Converts speech → text
* Responds using text-to-speech

### 🤖 AI-Powered Intelligence

* Understands natural language
* Chat like ChatGPT
* Maintains conversation context

### 🎬 YouTube Automation

* “Play Arijit Singh songs”
* Opens YouTube
* Searches
* Plays first video automatically

### 🔍 Smart Google Search

* Searches queries
* Opens first result automatically

### 🖥️ System Control

* Open apps dynamically
* Example:

  * “open notepad”
  * “open chrome”

### 🧠 Smart Decision Engine

* Decides:

  * Action ⚙️ OR Chat 🤖
* No hardcoded commands needed

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/PAURUSH077/SAQIB-Smart-Automated-Query-Intelligent-Bot.git
cd SAQIB-Smart-Automated-Query-Intelligent-Bot
```

---

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file:

```
GEMINI_API_KEY=your_api_key_here
```

---

### 4️⃣ Run the Project

```bash
python main.py
```
